# GaoYouGanqu

高邮灌区遥感分类、耕地识别和灌溉分析代码。

## 版本管理

项目采用 Git 分支、独立提交、语义化标签和 GitHub Release 管理版本。
详细规则见 [VERSIONING.md](VERSIONING.md)。

当前主要脚本：

- `scripts/gaoyou_landcover_rf_2020.js`：基础随机森林土地利用分类。
- `scripts/gaoyou_landcover_hierarchical_rf_2020.js`：增强版分层随机森林分类。
- `scripts/gaoyou_irrigation_2020.js`：灌溉稻田初步识别。
